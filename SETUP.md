# 🚀 セットアップガイド

このドキュメントでは、匿名掲示板「AnonBBS」をローカル環境で動作させるまでの手順を詳しく説明します。

## ステップ1: Node.jsのインストール

### Windows

1. [Node.js公式サイト](https://nodejs.org/ja/)にアクセス
2. **LTS版**（推奨版）をクリックしてダウンロード
3. ダウンロードした`.msi`ファイルをダブルクリック
4. インストールウィザードの指示に従ってインストール
5. **PowerShellまたはコマンドプロンプトを再起動**

### macOS

1. [Node.js公式サイト](https://nodejs.org/ja/)からLTS版をダウンロード
2. `.pkg`ファイルをダブルクリックしてインストール
3. ターミナルを再起動

### インストール確認

PowerShell、コマンドプロンプト、またはターミナルで以下のコマンドを実行：

\`\`\`bash
node --version
# 出力例: v20.11.0

npm --version
# 出力例: 10.2.4
\`\`\`

バージョン番号が表示されればインストール成功です！

## ステップ2: プロジェクトディレクトリに移動

\`\`\`bash
cd C:\\Users\\PCuser\\5ch-bbs
\`\`\`

または、エクスプローラーでプロジェクトフォルダを開き、アドレスバーに`cmd`と入力してEnterを押すことでも開けます。

## ステップ3: 必要なパッケージをインストール

プロジェクトディレクトリで以下のコマンドを実行します：

\`\`\`bash
npm install
\`\`\`

このコマンドは、`package.json`に記載されているすべての依存パッケージをインストールします。
初回は5〜10分程度かかる場合があります。

### インストールされるパッケージ

- **Next.js**: Reactフレームワーク
- **React**: UIライブラリ
- **TypeScript**: 型安全な開発環境
- **Tailwind CSS**: CSSフレームワーク
- **better-sqlite3**: SQLiteデータベース
- **SWR**: データ取得ライブラリ
- **Framer Motion**: アニメーションライブラリ
- **Lucide React**: アイコンライブラリ
- **shadcn/ui**: UIコンポーネント

## ステップ4: 開発サーバーの起動

\`\`\`bash
npm run dev
\`\`\`

成功すると、以下のようなメッセージが表示されます：

\`\`\`
- ready started server on 0.0.0.0:3000, url: http://localhost:3000
- event compiled client and server successfully
\`\`\`

## ステップ5: ブラウザでアクセス

ブラウザを開いて、以下のURLにアクセス：

**[http://localhost:3000](http://localhost:3000)**

匿名掲示板のトップページが表示されれば成功です！🎉

## 📱 使い方

### スレッドを作成する

1. トップページ右側の「新規スレッド作成」フォームに入力
2. 「スレッドタイトル」を入力（必須）
3. 「名前」を入力（省略可。未入力時は「名無しさん」）
4. 「スレッドを立てる」ボタンをクリック

### レスを投稿する

1. スレッド一覧から見たいスレッドをクリック
2. スレッド詳細ページが開きます
3. 下部の「レスを書く」フォームに入力
4. 「本文」を入力（必須）
5. 「投稿する」ボタンをクリック

## 🛠️ トラブルシューティング

### ポート3000が既に使用されている

エラーメッセージ: `Port 3000 is already in use`

**解決方法**:

1. 別のポートを使用する：
   \`\`\`bash
   npm run dev -- -p 3001
   \`\`\`
   その場合は `http://localhost:3001` でアクセス

2. または、既存のプロセスを終了する（Windows）：
   \`\`\`bash
   netstat -ano | findstr :3000
   taskkill /PID <PID番号> /F
   \`\`\`

### npm install でエラーが出る

**原因**: Node.jsのバージョンが古い可能性

**解決方法**:
1. Node.jsを最新のLTS版に更新
2. PowerShellを再起動
3. 再度 `npm install` を実行

### データベースファイルが作成されない

**解決方法**:
- プロジェクトディレクトリの書き込み権限を確認
- 開発サーバーを一度停止（Ctrl+C）して再起動

### ページが表示されない

**確認事項**:
1. 開発サーバーが起動しているか確認（ターミナルを確認）
2. ブラウザのURLが `http://localhost:3000` になっているか確認
3. ブラウザのキャッシュをクリア（Ctrl+Shift+Delete）

## 🔄 開発サーバーの停止

開発サーバーを停止するには、ターミナルで **Ctrl+C** を押します。

## 📦 本番ビルド

本番環境用にビルドする場合：

\`\`\`bash
npm run build
npm start
\`\`\`

## 🆘 サポート

問題が解決しない場合は、以下を確認してください：

1. Node.jsのバージョン（18.x以上）
2. npm のバージョン（9.x以上）
3. プロジェクトディレクトリが正しいか
4. `package.json` が存在するか
